# 🎤 INTER-VU

An AI-powered mock interview platform that helps users prepare for interviews with real-time feedback.

## 🚀 Features

- 📄 Resume Upload (PDF/DOCX)
- 🤖 AI-Generated Interview Questions
- 👁️ Eye Tracking using MediaPipe
- ✋ Hand Gesture Detection
- 🎥 Live Camera Monitoring
- 📊 Interview History
- 🌐 Modern Web Interface

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- Flask

### AI & Vision
- Google Gemini API
- MediaPipe
- OpenCV

### Database
- SQLite

## ▶️ How to Run

### Backend

```bash
cd backend
python app.py
```

### Frontend

```bash
cd frontend
python -m http.server 8000
```

Then open:

```
http://localhost:8000
```
